## HACS Store Icon

The local logo improves the repository and HACS detail page. The small icon shown in HACS search and list views may still use HACS or Home Assistant brand metadata. When submitting this card to default HACS, also submit the logo to the relevant brand metadata process if HACS requests it during review.

Use the SVG in this repository as the source artwork. The included PNG is 512 px and can be resized if a brand submission requires another raster size.
